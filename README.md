# VeriTuition MVP

This repository contains **VeriTuition**, a confidential college fee & scholarship payment system built for the Team1 India SpeedRun Hackathon.

## Structure
- `/verituition` - The main project folder containing the Next.js frontend and Hardhat smart contracts.

Please see the [VeriTuition README](verituition/README.md) for detailed instructions on how to run the project.

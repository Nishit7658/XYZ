# Builder Hub Submission Content

## Project Overview

VeriTuition is a privacy-preserving college fee and scholarship payment system on Avalanche. It hides sensitive wallet and payment details while still letting colleges verify that payments were made.

## Detailed Description

Traditional blockchain payments expose balances and transfers publicly. VeriTuition uses encrypted payment payloads and an auditor-controlled verification path so students can pay tuition privately and colleges can still reconcile payments.

## GitHub Description

VeriTuition is a privacy-preserving Avalanche dApp for secure college fee and scholarship payments. It combines encrypted payment flows, an auditor verification model, and a simple student dashboard built with Next.js and Solidity.

## Demo Script

2. Connect a wallet.
3. Mint test VERI tokens.
4. Enter a tuition amount and payment note.
5. Send the confidential payment.
6. Show the receipt screen.
